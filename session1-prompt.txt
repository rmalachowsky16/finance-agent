## Session goal: scaffold the Market Intelligence Agent

### What it does
- Downloads earnings call transcripts from SEC EDGAR
- Chunks transcripts to fit Claude's context window
- Sends chunks to Claude API for summarization
- Extracts: key themes, management tone, guidance revisions
- Returns: structured JSON + human-readable summary
